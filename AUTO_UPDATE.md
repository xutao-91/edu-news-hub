# 📰 教育新闻自动更新说明

## ⏰ 更新时间

**每天上午 10:00 CDT（美国中部时间）**

- Cron表达式: `0 15 * * *` (UTC时间，对应CDT 10:00)
- 自动推送到: https://xutao-91.github.io/edu-news-hub/

## 🔄 更新流程（真实新闻抓取）

1. **10:00 CDT** - Cron定时触发
2. **搜索新闻** - 使用Tavily API搜索115+教育新闻源
3. **智能筛选** - 筛选联邦政策+中西部11州相关新闻
4. **提取日期** - 从原文中提取发布日期
5. **分类整理** - 按类别整理（联邦/AI/高等教育/师资/国际生/STEM）
6. **更新GitHub** - 自动提交并部署

## 📡 数据来源

| 来源 | 说明 |
|------|------|
| **Tavily API** | AI搜索引擎，实时抓取教育新闻 |
| **搜索关键词** | 联邦政策、AI教育、高等教育、教师短缺、国际生签证、STEM等 |
| **筛选范围** | 联邦政策全收录 + 中西部11州新闻 |
| **链接验证** | 所有URL均为真实可点击的新闻链接 |

## 🛠️ 技术实现

| 组件 | 说明 |
|------|------|
| `fetch_real_news.py` | 真实新闻抓取脚本 |
| `auto_update_news.sh` | 自动更新主脚本 |
| `Tavily API` | 新闻搜索API |
| `GitHub Actions` | 自动部署 |

## 📝 手动触发

如需立即更新，运行：
```bash
/root/.openclaw/workspace/scripts/auto_update_news.sh
```

## 📊 更新记录

```bash
tail -f /root/.openclaw/workspace/edu-news-update.log
```
